Admin access
This app is a single HTML file. No server software is required for basic use. To share it across your office network or remotely, follow the options below.

Option 1 — Local network (office / same Wi-Fi) · Recommended

Host the app on one Windows PC so everyone on the same network (office LAN or Wi-Fi) can open it on any device — Windows laptops, Android phones, tablets.

Step-by-step setup on the host PC
Android access: On any Android phone connected to the same Wi-Fi, open Chrome and type the URL shown above (e.g., http://192.168.1.x:8000). The app opens instantly — no installation needed.

Option 2 — Remote access from other locations

For staff working from home, field agents, or accessing from Cebu/Manila/other offices.

Angrok (easiest, free)

Tunnels your local server to a public URL in seconds. No router config needed.

BGitHub Pages (free, permanent)

Host the HTML file permanently online. Accessible anywhere, any device, always-on.

CVPN (most secure)

Connect remote users to the office network as if they were on-site.

DRouter port forwarding

Forward port 8000 on your router to the host PC. Staff access via your public IP.

Android — add to home screen (app-like experience)

Once the URL is open in Chrome on Android, the app can be pinned to the home screen so it opens like a native app — full screen, no browser bar.

Database & storage

All quotation and client data is stored in Google Sheets and Google Drive. Updating the app code never affects this data — it persists permanently in your Google account.

Shared Drive folder Admin sets once — all users inherit

All users must save quotation files to one shared Google Drive folder. Create a folder in Google Drive, share it with every user as Editor, then paste its ID here and click Save settings. Other users will use this folder automatically when they sign in.
How to find the folder ID: Open the folder in Google Drive → the ID is the last part of the URL: drive.google.com/drive/folders/THIS-PART
